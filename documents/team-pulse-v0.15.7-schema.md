# Team Pulse schema

Current schema version: 2
Generated by Team Pulse v0.15.7

No durable-schema changes were introduced in v0.15.7. v0.15.7 is a scanner hygiene release: the built-in ZIP archive builder was removed, seven orphan functions were deleted, six innerHTML sites were rewritten to textContent/createElement, and the CSS block was whitespace-compressed. None of those changes touch the durable document shape or the event model.

Important runtime note: the open direct-report workspace continues to keep an in-memory draft of unsaved form edits so re-renders do not wipe the visible form. That draft state is UI-only and is not written into the Team Pulse JSON until the direct report workspace is explicitly saved.

## Event types

- **person_created** — Records a new person being created.
- **person_updated** — Records profile changes for an existing person.
- **note_added** — Records a new notes snapshot for a person.
- **meeting_logged** — Records a meeting touchpoint.
- **meeting_deleted** — Records deletion of a previously logged meeting.
- **person_archived** — Records archival/deletion of a person.

## Migration history

- **v0 -> v2** — Legacy flat app state and pre-schema JSON migrate into the durable event-backed document.
- **v1 -> v2** — Schema v1 wrapped state migrates into the durable event-backed document.
